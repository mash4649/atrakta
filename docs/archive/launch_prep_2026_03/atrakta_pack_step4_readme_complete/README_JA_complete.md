# Atrakta

> **インターフェースに縛られないAIコーディング**

Cursor · Claude Code · Codex CLI · VS Code · aider · 将来のAIツール

Atrakta は AI開発を

**安全・再現可能・復帰可能・可観測**

にする **AIコーディングランタイム** です。

------------------------------------------------------------------------

## ⭐ なぜAtrakta？

AIコーディングツールは急速に進化しています。

しかしツールを変えると開発フローが壊れる問題があります。

  問題                   起きること
  ---------------------- ----------------------------
  インターフェース依存   特定エディタにロックイン
  セッション不安定       AIの挙動が毎回変わる
  デバッグ困難           自動化失敗の原因が追えない
  危険な変更             AIが破壊的変更をする

Atrakta は

**AI開発のランタイムレイヤー**です。

> AIコーディングのための Git

------------------------------------------------------------------------

## 🚀 主な特徴

### ツール非依存

Cursor / Claude / CLI

どのツールでも同じ開発体験。

------------------------------------------------------------------------

### AIセッション復元

    atrakta resume

途中からAI作業を再開できます。

------------------------------------------------------------------------

### 決定論的パイプライン

    Detect → Plan → Apply

------------------------------------------------------------------------

### AI操作ログ

    .atrakta/events.jsonl

すべてのAI操作を記録。

------------------------------------------------------------------------

## ⚡ クイックスタート

### macOS / Linux

    curl -fsSL https://raw.githubusercontent.com/afwm/Atrakta/main/scripts/install.sh | bash
    atrakta init --interfaces cursor
