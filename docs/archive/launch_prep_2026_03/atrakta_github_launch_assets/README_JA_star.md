## ⭐ なぜAtrakta？

AIコーディングツールは急速に進化しています。

しかし実際の開発では次の問題が起きます。

  問題                   起きること
  ---------------------- ----------------------------
  インターフェース依存   特定ツールにロックイン
  セッション不安定       AIの挙動が毎回変わる
  デバッグ困難           自動化失敗の原因が追えない
  危険な変更             AIが破壊的変更をする

Atraktaは

**AI開発のランタイムレイヤー**です。

> AIコーディングのためのGit

------------------------------------------------------------------------

## 🚀 主な特徴

### ツール非依存

Cursor / Claude / CLI

どのツールでも同じ開発体験

### AIセッション復元

    atrakta resume

AI作業を途中から完全復元

### AI操作ログ

    .atrakta/events.jsonl
