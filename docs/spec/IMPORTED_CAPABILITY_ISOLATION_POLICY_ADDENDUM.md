# IMPORTED_CAPABILITY_ISOLATION_POLICY_ADDENDUM

## Default Isolation Policy
For imported capabilities before review:
- host access: default deny
- full network access: default deny
- full secrets access: default deny
- executable status: default false

## Approval Conditions
Explicit approval is required before any promotion step that increases trust.

## Notes
- This addendum does not replace existing isolation/security model.
- It tightens imported capability defaults only.
