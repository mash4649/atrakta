# 同期ポリシー

[English](../../en/02_仕様/04_同期ポリシー.md) | [日本語](./04_同期ポリシー.md)


## sync level

- `0`: 同期提案を行わない（既定）
- `1`: proposal-only（提案を生成、適用は承認必須）
- `2`: strict（AGENTS由来提案を意思決定に使わない）

`--sync-level` が未指定の場合は `ATRAKTA_SYNC_LEVEL` を参照します。

## proposal-only の原則

- AGENTSの指示は即時反映しない。
- `doctor --sync-proposal` で差分を可視化する。
- `doctor --apply-sync` で明示承認後に反映する。

## 反映対象（現行）

- `hints.prefer`
- `hints.disable_interfaces`

未サポートinterface IDは自動的に除外されます。
